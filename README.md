## 🛠️ 系统要求

- Windows 10/11 (64位)
- NVIDIA显卡 (最低4GB显存)
- 驱动版本 >= 512.15
- 8GB+ 系统内存
- SSD建议预留20GB空间

## 📦 快速开始

1. 下载并解压软件包
2. 运行 LightningMiner.exe
3. 输入您的DOGE钱包地址
4. 选择挖矿设备和矿池
5. 一键启动挖矿
